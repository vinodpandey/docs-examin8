**Examin8 Test Generator** is available as both an **Android app (**[**Examin8 App**](https://play.google.com/store/apps/details?id=com.examin8.exam)) and a web platform at https://examin8.com. Both versions offer the same user interface and features for a seamless experience across devices.

## Get started in three steps

Setup your institute/school/coaching center details and start creating testpaper in minutes.

### Step 1: Login to Examin8

<AccordionGroup>
  <Accordion title="Access website or mobile app" icon="copy">
    Download the [Examin8 App](https://play.google.com/store/apps/details?id=com.examin8.exam) or access the website [examin8.com](http://examin8.com) and click [**Start Now**](https://app.examin8.com/).

    <Tip>
      If you are using android phone, we recommend downloading the [Examin8 App](https://play.google.com/store/apps/details?id=com.examin8.exam). iPhone users can access the website [examin8.com](http://examin8.com) in Safari or Chome browser.
    </Tip>
  </Accordion>
  <Accordion title="Login using your preferred method" icon="rectangle-terminal">
    We support below methods for login.

    - Phone number (recommended)
    - Google 
    - Facebook
    - Email

    If you want to create a new account with your email ID, click **Sign in with Email** and then click **Sign up**.

    <Tip>
      We recommend login with phone number as it is easier and faster to login with OTP. No need to remember login email/password.
    </Tip>
    ![image.png](/images/image.png)
  </Accordion>
</AccordionGroup>

### Step 2: Enter Institute Details

<AccordionGroup>
  <Accordion title="Install our GitHub app" icon="github">
    Install the Mintlify GitHub app from your [dashboard](https://dashboard.mintlify.com/settings/organization/github-app).

    Our GitHub app automatically deploys your changes to your docs site, so you don't need to manage deployments yourself.
  </Accordion>
  <Accordion title="Update your site name and colors" icon="palette">
    For a first change, let's update the name and colors of your docs site.

    1. Open `docs.json` in your editor.
    2. Change the `"name"` field to your project name.
    3. Update the `"colors"` to match your brand.
    4. Save and see your changes instantly at `http://localhost:3000`.

    <Tip>
      Try changing the primary color to see an immediate difference!
    </Tip>
  </Accordion>
</AccordionGroup>

### Step 3: Go live

<Accordion title="Publish your docs" icon="rocket">
  1. Commit and push your changes.
  2. Your docs will update and be live in moments!
</Accordion>
