## **Overview**

Examin8 allows users to generate test papers in various formats, including PDF and Word. You can utilize different modes such as Express, Automated, and Manual, making it adaptable for various educational contexts.

## Create testpapers in seconds

Create test papers that are fully aligned with the latest syllabus.

<Columns cols={2}>
  <Card title="Express Mode" icon="pen-to-square" href="/development">
    Select questions for each chapters. Suitable if less customization required.
  </Card>
  <Card title="Dynamic Mode" icon="palette" href="/essentials/settings">
    Select the chapters, and the paper will be generated automatically using a prefilled blueprint.
  </Card>
  <Card title="Automated Mode" icon="map" href="/essentials/navigation">
    The question paper will be generated automatically based on a predefined blueprint, including specified chapters and chapter-wise weightage.
  </Card>
  <Card title="Manual Mode" icon="terminal" href="/api-reference/introduction">
    Add questions one by one from each chapter. Suitable if too much customization required.
  </Card>
  <Card title="I will Add Questions" icon="terminal" href="/api-reference/introduction">
    Add your own questions or select questions from your own question bank.
  </Card>
  <Card title="NCERT Textbook" icon="terminal" href="/api-reference/introduction">
    Add questions from NCERT Textbooks. Best fit for creating question papers from NCERT only.
  </Card>
